### 相册布局
* 统一写成一个class格式,然后通过js对class内的元素进行操作
* js 和 css 文件以第三方脚本引入的方式引入
### 拼图布局
* 主要对照片dom进行重塑
* -webkit-clip-path 裁剪的用法
### 瀑布布局
* 通过用户提供的容器大小来初始化布局
* 计算每列的总高度，然后选取填入的相框
### 木桶布局
算法如下：
* 初始化数据： 初定每行高度为200px，初始化每张图片的长高比并记录，根据长宽比计算出宽度；
* 图片分组： 当某张以原定200px的高度计算出来的宽度大于要插入行的总宽度，这时把已经插入的照片看成一张照片，扩大到把一行占满，计算出实际高度；
* 实际高度乘以每张照片的长高比计算出实际宽度即可，此时这几张照片为一组；
* 以此类推，剩下几张照片不够一组按200px处理放在最后一行即可；
* 刷新dom
